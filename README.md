# Expense Tracker

A full-stack MERN Expense Tracker application.

## Features

- User Authentication
- Add/Edit/Delete Expenses
- Expense Categorization
- Secure JWT Authentication

## Tech Stack

- React + Vite
- Node.js
- Express.js
- MongoDB
- JWT
- Render
- Vercel
